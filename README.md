# Password-Generator-🔐 

A simple and powerful **Password Converter** built with **HTML**, **CSS**, and **JavaScript**.  
This tool allows users to convert any password into **Base64** or **SHA‑256**, and also checks the **password strength** (Weak, Medium, Strong).

Perfect for beginners learning encryption basics, hashing, DOM manipulation, and interactive UI design.



## 🚀 Features

### 🔑 Conversion Tools
- Convert password to **Base64**
- Convert password to **SHA‑256 hash**
- Instant results

### 🛡 Password Strength Checker
- Weak  
- Medium  
- Strong  
- Based on length, uppercase letters, and numbers

### 🎨 UI Features
- Clean and modern interface  
- Responsive layout  
- Easy-to-read output boxes
